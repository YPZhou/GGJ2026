# GGJ2026
Global Game Jam 2026

![游戏封面](https://github.com/user-attachments/assets/66db22ac-2a8e-41a8-97cb-db1b3a078edc)

《小猫咪看不得这些！！》是一款考验你瞬时反应与预判的快速游戏。  
扮演一名守护者，在电视画面变糟的瞬间，挡住猫咪好奇的视线。  
小心，猫咪的闪避能力超乎你的想象。  
保护好它，否则你会收获一个……心灵受伤的主子。

玩法说明：  
绝不能让小猫咪看到不该看的画面！  
撑过30秒就是胜利。

操作说明：  
WSAD 移动手掌

策划：黑白色的枫  
程序：Cataphract、超大型呆萌  
美术：Lucy、禾十、清水因、信使来送信、真目菌  

场外援助：石榴大师、摊主  
卧底：PH777
